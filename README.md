# Getting started 

1. Install [rcm](https://github.com/thoughtbot/rcm#installation):

    ```bash
        sudo add-apt-repository ppa:martin-frost/thoughtbot-rcm
        sudo apt-get update
        sudo apt-get install rcm 
    ```

2. Install the dotfiles:

    ```bash
        env RCRC=$HOME/.dotfiles/rcrc rcup
    ```
